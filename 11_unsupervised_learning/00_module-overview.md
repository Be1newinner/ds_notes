# Module 11: Unsupervised Learning

## What Students Will Learn
- The core principles of unsupervised learning and how it differs from supervised learning.
- How to group similar data points together using various clustering algorithms.
- The intuition, mathematics, and implementation of K-Means, Hierarchical Clustering, DBSCAN, and GMM.
- How to evaluate clustering models using metrics like Silhouette Score and the Elbow Method.

## Why This Module Matters
In the real world, most data is unlabeled. Unsupervised learning is the primary tool for finding hidden patterns, segmenting customers, discovering taxonomies, and detecting anomalies without relying on predefined labels.

## Prerequisites
- Python for Data Analysis (Pandas, NumPy)
- Python Visualization (Matplotlib, Seaborn)
- Basic Business Statistics (Mean, variance, probability distributions)
- Distance metrics (Euclidean, Manhattan)

## Teaching Sequence
1. **Clustering Fundamentals**: What is clustering, when to use it, and distance metrics.
2. **K-Means & Variants**: The workhorse of clustering. Intuition, algorithm, and finding K.
3. **Hierarchical Clustering**: Building taxonomies and interpreting dendrograms.
4. **Density-based Methods (DBSCAN)**: Handling non-spherical clusters and outliers.
5. **Gaussian Mixture Models (GMM)**: Probabilistic clustering and overlapping groups.

## Main Subtopics
- Clustering Fundamentals
- K-Means and Variants
- Hierarchical Clustering
- Density-based Methods
- Gaussian Mixture Models

## Real-world Use Cases
- Customer segmentation for targeted marketing.
- Document clustering for topic discovery.
- Image compression and color quantization.
- Fraud and anomaly detection.
- Genomic clustering in bioinformatics.

## Expected Outcomes
By the end of this module, students will confidently identify when to use unsupervised learning, select the appropriate algorithm for their data's shape, implement the model using scikit-learn, and present business value from the discovered clusters.
