Activation Functions

A comprehensive study guide on non-linearity, neuron mechanisms, and modern optimization techniques.

Executive Summary

This guide explores the fundamental role of activation functions in neural networks. Without them, a neural network is just a linear regression model. We cover Sigmoid, tanh, ReLU, and Leaky ReLU, along with their mathematical derivatives which are crucial for backpropagation.

Technical Deep Dive: Math & Gradients

Sigmoid ($\sigma$)

$\sigma(x) = \frac{1}{1 + e^{-x}}$

Derivative: $\sigma(x)(1 - \sigma(x))$

Pro: Range (0,1). Con: Vanishing Gradients.

Tanh

$tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$

Derivative: $1 - tanh^2(x)$

Pro: Zero-centered. Con: Vanishing Gradients.

ReLU

$f(x) = max(0, x)$

Derivative: $1$ if $x > 0$, else $0$

Pro: Fast, no vanishing gradients for $x>0$. Con: Dying ReLU.

Leaky ReLU

$f(x) = max(0.01x, x)$

Derivative: $1$ if $x > 0$, else $0.01$

Pro: Fixes "Dying ReLU" problem.

Technical Execution

import numpy as np

# Numpy vectorized implementations
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def relu(x):
    return np.maximum(0, x)

def leaky_relu(x, alpha=0.01):
    return np.where(x > 0, x, alpha * x)

# Example of "Dying ReLU" visualization
inputs = np.array([-10, -5, 0, 5, 10])
print(f"ReLU Output: {relu(inputs)}") 
# Result: [0 0 0 5 10] - Notice the first three inputs are "dead"
