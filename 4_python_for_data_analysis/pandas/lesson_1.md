## 1. Introduction to Pandas and Data Analysis

### What is Pandas?
Pandas is a Python library for working with structured data in tables and time-based datasets. It is one of the core tools used in data analysis, data cleaning, and data preparation for machine learning and reporting. [pandas.pydata](https://pandas.pydata.org/docs/user_guide/10min.html)

### Why Pandas is used
- It makes data handling fast and readable.
- It helps clean messy datasets.
- It supports powerful filtering, grouping, merging, and summarizing.
- It works well with CSV, Excel, SQL, JSON, and many other data sources. [datacamp](https://www.datacamp.com/tutorial/pandas)

### Where Pandas fits in data science
Pandas is usually used after data collection and before modeling or visualization. It helps transform raw data into a form that can be analyzed with tools like NumPy, Matplotlib, Seaborn, scikit-learn, and machine learning pipelines. [youtube](https://www.youtube.com/watch?v=5U2bOjbZ4B0)

### Main strengths
- Easy syntax for real-world data tasks.
- Built around labeled data structures.
- Excellent support for missing data.
- Strong integration with Python’s data ecosystem. [pandas.pydata](https://pandas.pydata.org/docs/dev/whatsnew/v3.0.0.html)

### Core data structures
- Series: a one-dimensional labeled array.
- DataFrame: a two-dimensional labeled table. [youtube](https://www.youtube.com/watch?v=ZnCvwv-cxlQ)

### Typical use cases
- Sales and business reporting.
- Financial analysis.
- Web analytics.
- Data cleaning and preprocessing.
- Feature engineering for machine learning.
- Exploratory data analysis. [pandas.pydata](https://pandas.pydata.org/docs/getting_started/intro_tutorials/index.html)

### Basic workflow
1. Load data from a file or database.
2. Inspect the dataset.
3. Clean and transform values.
4. Filter, sort, and group records.
5. Summarize results.
6. Export the cleaned output. [pandas.pydata](https://pandas.pydata.org/docs/user_guide/10min.html)

### What students should learn first
- How pandas is imported.
- What Series and DataFrame mean.
- How to read and inspect data.
- How to select rows and columns.
- How to handle missing values.
- How to perform simple analysis on a dataset. [youtube](https://www.youtube.com/watch?v=ZnCvwv-cxlQ)

### Simple example ideas
- Load a CSV of student marks and find top performers.
- Analyze monthly sales and total revenue.
- Clean a customer list with missing emails.
- Group product data by category and compare averages.
- Filter records based on age, city, or score. [datacamp](https://www.datacamp.com/tutorial/pandas)
